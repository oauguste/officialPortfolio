Task Organizer

At the heart of the Task Organizer is a sophisticated yet user-friendly interface. Users are able to create, read, delete, update notes and folders. This application leverages HTML5, Tailwind CSS, and JavaScript. The design is responsive, ensuring an optimal viewing experience across a variety of devices. Visual aesthetics were also a priority - with meticulously chosen background images and color schemes contributing to a pleasant user interface.

Real-Time Weather Data App

An interactive and user-friendly application developed with React that provides real-time weather data of any city globally. The application leverages OpenWeatherMap API to fetch the necessary data and uses axios for handling HTTP requests. This project demonstrates proficiency in React, JavaScript, asynchronous programming, API integration, and error handling. The application is designed with a focus on usability, responsiveness, and real-time data retrieval.

Dynamic Restaurant Menu

This web application serves as a dynamic online menu for a restaurant. It is a responsive, full-stack web development project that allows users to view different restaurant menus. The code is well-structured and uses modern JavaScript practices. The project could be expanded in the future to include server-side programming for actual order placement and processing. This application leverages HTML5, Tailwind CSS, and JavaScript for front-end development.
